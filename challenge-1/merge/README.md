# Cauchy problem solver based on the theta method

Team: Francesca Bettinelli, Massimo Rizzuto

Compile: make

Execute: ./main

Output:

- "result.txt": file containing the timesteps and the approximate solution
- "error_conv.txt": file containing the convergence analysis of the error

Plot the output (saved in "plots"): ./plot_script.sh

Remove the object files: make clean

Remove the object files and the executable: make distclean
